Available Routes

TYPE ROUTE INPUT OUTPUT DESCRIPTION
LOGIN [POST]/api/login {email:'',password:''} {token:''} Accepts email and password and return json token
[GET]/api/login [header]x-auth-token:'' {userModel} Accepts token as header and returns the user data
REGISTER [POST]/api/register {name:'',email:'',password:''} {token:''} Accepts name,email and password and return json token
